Counters to replace the start attribute

To save using a class, I’ve used a nth-of-type selector for the second list. You’ll need to use a modern browser for this example to work. For production web sites use a class name until web browser support catches up. For this particular example, reseting the counter on the body element would have the same effect, without the counter offset, as the counter doesn't reset when you start a new list. However, If you wanted a third list to start at 1 again, it wouldn't work using that approach, as it would continue counting from 8.
